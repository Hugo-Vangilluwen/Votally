# Votally

Votally is a voting application in local network.

This is a work in progress.

Do not use for major votes.

## Build
```bash
cd votally-cli
cargo build
```

## Run
As a server:
```bash
cd votally-cli
cargo run -- --server optionA optionB ...
```

As a client:
```bash
cd votally-cli
cargo run
```
